Want to create your own web site? Not sure where to begin? Afraid it might be expensive? Let Dave show you where to find free web design software, free web server space, and a free web-based e-mail address.

Then Dave will guide you step-by-step through the web site creation process. Begin with a simple page. Add links, color, images, a background, animation, a message banner, and sound. Learn how to use tables to achieve a professional layout.

Transform your simple page into a multi-page web site. Add an image map, a visitor counter, a guest book, and a password-protected area. Convert to a free easy-to-remember web site address. Then register your web site with the key search engines.

Dave's Quick 'n' Easy Web Pages is used by North American educators from fifth grade through college. Great for small business owners too.
Dave Lindsay got his start as a web designer in 1996 when he was eleven years old. It was then that he launched The Official Redwall Web Site for Brian Jacques, British author of the Redwall adventure series for young people.

Visitors from over 135 countries enjoy the Redwall Abbey Web Site, http://www.redwall.org.

Dave is now studying computer engineering at the University of Calgary. As part of his studies, he is completing a six month internship with Microsoft in Redmond, Washington.
